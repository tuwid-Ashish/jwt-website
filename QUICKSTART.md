## 💡 Tips & Tricks

### Keyboard Navigation
- **Tab** → Move between links/buttons
- **Enter** → Activate links/buttons
- **Escape** → Close mobile menu
- **Smooth Scroll** → Click anchor links

### Browser Shortcuts
- **Ctrl+Shift+I** (Windows) or **Cmd+Option+I** (Mac) → Open DevTools
- **F12** → Full screen DevTools
- **Ctrl+A** (Cmd+A) → Select all
- **Ctrl+Shift+Delete** (Cmd+Shift+Delete) → Clear cache

### Customization
All colors, fonts, and spacing are defined in `styles.css` as CSS variables:
```css
:root {
  --color-primary: #0d3d5c;
  --color-accent: #d4860e;
  --font-display: 'Instrument Serif', Georgia, serif;
}
```

Change these values to rebrand instantly!

## ❓ FAQ

**Q: Can I edit the content?**
A: Yes! Open any `.html` file in a text editor and update the content. Save and refresh browser.

**Q: How do I add new pages?**
A: Create a new `.html` file, copy the header/footer from an existing page, and add your content.

**Q: What about the contact form?**
A: The form validates data but doesn't submit to a server. To make it functional, you'll need a backend service (like Formspree, Netlify Forms, or your own API).

**Q: Can I change colors?**
A: Yes! Edit the CSS custom properties in `styles.css` lines 1-50.

**Q: Is it mobile-friendly?**
A: Absolutely! Fully responsive from 320px (small phones) to 2560px (large monitors).

**Q: Does it work offline?**
A: Yes! Except the contact form submission (which needs internet) and Google Fonts (which needs CDN).
